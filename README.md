# HanlonDiagonal pipeline

#### Overview
This is a function for generating diagonal CellenONE dispensing profiles of cuustom size

#### Quick use
Run `Rscript hanlonDiagonal.R [num of indexes] [num of columns] [num of rows]` on command line
